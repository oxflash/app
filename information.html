HOW IT WORKS

About

OX Flash is a btc flashing app to send fake bitcoin to the blockchain network, this transactions have a time limit, using some computing power to keep it onchain, amount of power used to keep transaction onchain depends on the duration, the more the duration the more then computing power needed. Confirmations can aslo be added to trick any node to make it look valid, currently amount of confirmations we provide for transactions is 12 confirmations, as customs nodes are needed to keep transactions on chain before the blochain network detects it.

Features

Daily Limit:
A 1 Bitcoin daily limit is set for users, the balance is automatically set back to 1 Bitcoin after 24hours
Duration
How long transactions stays on chain can be set by user upon making a transaction, 24hours, 48hours, 7days, 20days, 1 month.
Receiver Address Type
The wallet you're sending the transaction to is the wallet type which is listed in the app, inlcudes Trust wallet and Binance, there is a chance you're sending to an address you don't know, then you can select the Default walllet option.
Confirmations
Transactions can get upto 12 confirmations and minimum of 2 confirmations
